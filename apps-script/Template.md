## 3. 플레이스홀더 목록

다음 플레이스홀더가 Apps Script에서 자동으로 치환됩니다:

- `{{회사명}}` → data.companyName
- `{{대표자}}` → data.name
- `{{사업자등록번호}}` → data.businessNumber
- `{{이메일}}` → data.email
- `{{연락처}}` → data.phone
- `{{희망과제}}` → data.desiredSupport
- `{{사업아이디어}}` → data.businessIdea
- `{{목표시장}}` → data.targetMarket
- `{{경쟁력}}` → data.competitiveness

## 4. 템플릿 ID 확인

1. 템플릿 문서 생성 완료 후 URL 확인
2. URL 형식: `https://docs.google.com/document/d/[TEMPLATE_DOC_ID]/edit`
3. `[TEMPLATE_DOC_ID]` 부분을 복사
4. `Code.gs`의 `CONFIG.TEMPLATE_DOC_ID`에 설정

## 5. 스타일 가이드

- **제목**: 굵게, 크기 16pt
- **구분선**: ━━━━━ 사용
- **본문**: 일반, 크기 11pt
- **강조**: 굵게 또는 색상 사용

## 6. 고급 템플릿 (선택사항)

기존 `index.html`을 Google Docs로 변환하여 더 상세한 템플릿을 만들 수 있습니다:

1. `index.html`을 브라우저에서 열기
2. 인쇄 → PDF로 저장
3. Google Drive에 업로드
4. 우클릭 → "Google 문서로 열기"
5. 플레이스홀더 추가하여 템플릿화
